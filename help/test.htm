This class requires either RealAudio or RealVideo. Your browser must support JAVA.

This area will help you get your browser configured properly and get the proper software installed on your system. For RealAudio and RealVideo test all possible ways. You will be best off if you can do actual streaming. If you can only do Donwload-then-play then you will have to "flip" the slides manually. Generally if download does not work, then streaming and pseudo-streaming will not work. If you can do RealVideo, then you also should be able to do RealAudio.

  • Test RealVideo
    • Download-then-play - Slow but better than nothing - Auto slide coordination will not work
    • Pseudo-Streaming RealVideo - Fast but cannot skip around as easily
    • Streaming RealVideo
  • Test RealAudio
    • Download - then - play - Slow but better than nothing - Auto slide coordination will not work
    • True Streaming RealAudio - The COOLEST way - true random play capability
    • Pseudo-Streaming RealAudio - Fast but cannot skip around as easily - Will not work on RealAudio 2.0 - Requires RealAudio 3.0 or Higher
  • If you are enrolled in this class, test JAVA by entering the chatroom . (2 minutes)
  • If you are not enrolled in the class, JAVA is not necessary but if you are a glutton for punishment, you can test JAVA with a Dinosaur Matching game. (45 sec)

If you have passed the tests - Congratulations - Go take the class. You can convienently jump right to the lectures and get started!

If you have not passed the tests, install the missing software by following the instructions below.

Software Installation Area

Note - If you are using AOL, check here first for additional instructions.

  • Audio/Video
    • Intel system - go to RealVideo and install version 5.0 or better - if you have a sytem slower than a 486 you wont' be able to se the video.
    • Macintosh - RealAudio or RealVideo
    • America On-Line has its own set of problems for streaming and pseudo streaming, I need help developing documentation on how to do this through AOL. I have writen some documentation here.
    • You might also find the on-line documentation from RealAudio company helpful.
  • For JAVA, you may need to upgrade to a newer version of Netscape or Internet Explorer. You may have to enable JAVA execution on your browser. In Netscape go to Options/Network Preferences/Languages and Check "Enable JAVA". In Internet Explorer, there is a similar setting.
