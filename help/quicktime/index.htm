Installing Quicktime - EGR124

This is the step by step procedure for installing Quicktime EGR124 - Introduction to the Internet .

Macintosh systems usually come with Quicktime already installed but you may want to upgrade your Quicktime.

You may want to print out these instructions before starting.

  • Download the Quicktime by clicking here. Select "QuickTime Easy Install".

    Where you save the downloaded information is important. Go to the C:\ directory by clicking the "move up a level" one or more times.

  • If you have a C:\temp directory, skip this step. Once you have reached C:, create a directory called temp by clicking on the "Make New Directory" button. The directory name will initially be "New Folder" and the name will be highlighted so just type temp and
  • Then click to enter the temp folder to enter the C:\temp folder and click on "Save".
  • After the file has downloaded, exit Netscape or MS-Explorer. The installation will reconfigure your browser so it cannot be running.
  • Click on "My Computer" on your desk top. Click on "C:". Click on the "Temp" folder. You should see the file you just downloaded.
  • Begin the installation by clicking on the downloaded file qteasy.exe
  • During the installation you will be asked the following questions:
    • It will ask if you want to install Quicktime and the Plug-in - Say "Yes"
    • It will say Welcome - "Next"
    • If it asks you would like to reinstall Quicktime - say "Yes"
    • License terms - "Agree"
    • Do you want to install? - "Install"
    • Go ahead and scan for exsiting versions of Quicktime - "Start"
    • If it finds another Quicktime - Delete it - Quicktime has an anoying habit of scattering stuff all over your hard disk. Oh well.
    • Once the scan is done, click "Install"
    • Any strange questions which come up - Just say "OK"
    • When it says, "Updating .INI files", think to yourself - "Who was the moron who could not do a standard Windows-95 installation?" You don't need to type or click at this point.
    • When it says "Install Complete" - "OK" (It is not really done though)
    • After some flashing, it will come back with another license agreement - "Yes"
    • Select your browser - "Next"
    • If it asks you about searching a network drive - say "No" unless that is really where you have your browser stored.
    • It will ask where to install the plugin. "C:\...\Navigator\plugins" is just fine - "Next" - Note - It really means to say "C:\Program Files\Netscape\Navigator\plugins" in case you were interested.
    • The nifty bar graphs will bounce up and down and it will ask you to press "Finish" - so do it.
  • At this point, restart Netscape and try the EGR124 test again :)
  • You may delete the downloaded files qteasy32.exe in the C:\temp directory
  • If you want to remove this software, go to "Start" / "Settings" / "Control Panel" / "Add/Remove Programs". Find the software name in the list at the bottom of the Add/Remove Programs Dialog, select the software and click "Add/Remove".
