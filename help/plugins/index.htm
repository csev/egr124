Netscape Plug-ins Trouble Shooting

(If the images are overlaped on one another adjust the size of your Netscape window)

This page is designed to help people who have downloaded QuickTime, Realaudio, or Realplayer, but still are having problems with them either working at all, or automatically starting up from Netscape (i.e. Netscape doesn't require you to save the file then run it from Windows.)

I will start with QuickTime since when I downloaded it the installation did not set Netscape up so that QuickTime would run when I clicked on a QuickTime file.  First go to "Options" and click on "General Preferences" like this first image illustrates.  




You should now see something like the image to the left.  Click on the "Helpers" tab at the top, and scroll down until you see the "video/quicktime" position, which is also highlighted to the left.  If your window also doesn't have Play32.exe displayed as the program to run for a QuickTime movie like the images to the left, then click on the circle to the left of "Launch the Application:" and then click the "Browse" button. 



The Window to the right should pop up.  You need to find the Play32.exe file.  It should be located in your "Windows" directory.  To get to it, click on the triangle next to the "Look in" section of the window like the image to the right suggests.  Now click on the Windows95 (C:) icon.







Now find the "Windows" directory and click on it.  Scroll until you find the "Play32.exe" file and double click on it.  If the file is not in this directory make sure you did install QuickTime after you downloaded it.  If you are sure you did install it, then maybe install put the program in a different location than this.  Use Windows "Find" utility to locate the file and use the procedures above to go to the directory "Find" tells you the file is in.


If you successfully found the file and clicked on it, then the window that appears should look like the image to the right.  Make sure the "Launch the Application:" circle is marked and hit "OK."  If you want you can go to the CPS 291 Test Page, and try it out.












For RealPlayer make sure that you have the same setting as in the highlighted sections in the next two windows to the left.  If you don't, use similar procedures as described above for the QuickTime example.  The program should be at the following location:  C:\Real\Player\rvplayer.exe.  If you are using Realaudio instead of RealPlayer, then ware you see rvplayer in the two windows to the right, you should see raplayer.  It will be located at:  C:\Real\Player\raplayer.exe. 

Good Luck!!!
