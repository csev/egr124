How to Uninstall Programs in Windows 95

(If the images on this page over lap onto one another, adjust the size of your Netscape Window)

This page is designed to show a user how to use the built in uninstall feature in Windows 95.

In the following example I am uninstalling RealAudio from my computer, because I am upgrading to RealPlayer.  It is my experience that it is best to uninstall an old program before you upgrade to a new one.  This is because it eliminates problems you may encounter if you later try to uninstall the old program.  Sometimes parts of the new program are also removed if two programs are just different versions of the same thing.  The first step is to left mouse click on the "Start" button in you screens lower left-hand corner.  A menu will pop up like you see in the image to the right.  Move the mouse pointer up to "settings" and another menu pops up.  Go over to "Control Panel" and left mouse click on it.  The window below will come up.


The window that you see on your computer screen should be very similar to the image on the left.  Now locate the "Add/Remove Programs" icon, which is the highlighted icon in the image to the left.  Double click on this icon with your left mouse button.  Your computer should now display the window below.












Next, go to the "Install/Uninstall" tab at the top of the window.  Scroll down until you find the program you are interested in uninstalling from your computer.  In this example it is RealAudio Player 3.0.  Now highlight the correct program by left mouse clicking on it.  Now move the mouse pointer down to the "Add/Remove..." button like the image to the right shows and left click on it.  If you can not find the program you wish to uninstall, then Windows 95 must not be able to uninstall it.  Look in the instructions for the program for proper removal procedures, or on the original disk for an uninstall program.  If you still can not find a way to uninstall the program either leave it, or delete the files manually.






If you found the program to uninstall and clicked the "Add/Remove..." button, then the window to the left should pop up.  Click on "OK" and Windows 95 will begin removing the program.





When Windows 95 is done removing the program the window to the right will show up.  Click the "OK" button and you are done.






If you want, you can check and make sure the program is gone.  As you can see by the image to the left, the program is no longer listed.  For some programs, Windows 95 will not be able to uninstall all or any of the program.  If during the uninstalling process Windows 95 informs you that it can not remove all of the directories or files associated with a program you may want to manually remove them yourself.  Be careful when deleting files that you are removing the correct files, and not things that may be needed.  If you do accidentally remove a file that you need, check the "Recycle Bin" icon and restore the file.

Good Luck ! ! !
