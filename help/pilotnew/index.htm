Steps to Activating Pilot

First you must run telnet to connect to the pilot system. This can be done in the following ways:

  • On UNIX - telnet pilot.msu.edu
  • On a PC using FTP Software - tn pilot.msu.edu
  • On a PC Using Windows-95 - Start / Run / telnet pilot.msu.edu / "OK"
  • On a well-configured web browser, click here

At the login: prompt enter pilot. No password is required so just press the "Enter" key.

Then you will be prompted for your PID/SSN and PAN. The PAN will not appear when typed.

You may want to set up your forwarding address in Pilot if you already have an E-Mail account.

You can go through this process again if you forget the password but still have your PID and PAN.

Using the Pilot ID you can now enter the password protected portions of the class.
