The Ticketmaster vs Microsoft Dispute

  • Microsoft’s “Seattle Sidewalk” site on the Web

  • Includes concert, sporting event calendars

  • …and hyperlinks to Ticketmaster’s site to purchase those tickets

  • Ticketmaster objects, claiming a right to control how people enter their Web site

  • If visitors don’t see the front page, they don’t see all the ads

  • If Ticketmaster prevails, ominous implications for search engines
