Fiber Optic

  • Fiber Optic can be used in many ways

    • FDDI (Fiber Distributed Data Interface) - 1100Mb
    • ATM - Asynchronous Transfer Mode - 1Gb
    • Gigabit Ethernet - IEEE 802.3z - 1Gb

  • Fiber Optic tends to be used for long physical distance applications

    • Total fiber capacity is > Tb/sec 1012
    • Less susceptible to interference
    • Install once, then upgrade the equipment at either end to increase bandwidth
