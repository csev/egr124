Notes:

    Theory: If you cannot have fun with a technology it has a short life. At the University of Michigan there was a program called Confer which was a text based conferencing program. This program was both used for pleasure and technical discussions. One observation is that the enjoyable discussions served to make the users confortable with the environment so they were able to have excellent technical discussions.
    Network News (or Usenet News) started in 1979 now contains over 5000 topic areas with readers all over the world.
    The talk program allows instantaneous real-time chat between two users over the Internet..
    The FTP program allows filetransfer over a TCP/IP network. Through the use of anonymous (no account require) FTP programmers made a wide variety of software available over the Internet. This public domain software appraoch was often used to develop, deploy, and test a new piece of Internet software.
