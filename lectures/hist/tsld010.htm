Notes:

    The truly difficult technical challenge is to create reliable “connections” between programs on different computers assuming the intermediate network is unreliable. To accomplish this, data is broken into “Packets”. Each packet is independently routed through the network. If a packet is lost for one reason or another, it must be retransmitted by the original system. The network can detect the lost data, change its configuration to internally route packets to the destination using a redundant path. Because each packet is independent, when data is lost, it is only a small amount of data.
