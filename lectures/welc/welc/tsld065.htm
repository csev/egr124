Notes:


    It is important to be in the proper “local” directory and the proper “remote” directory “remote” directory before you transfer the files. For example, lets say you saved a picture to “me.gif” in the \temp\photos directory on your hard disk and you wanted to upload it to your main web space in the web subdirectory of your home directory on the server. It would work something like this:
    ftp pilot.msu.edu
    (log in)
    lcd \temp\photos
    cd web
    binary
    put me.gif
    quit
    Before the put command you moved into the proper directory on the hard disk, the proper directory on the server, and set the mode to binary before actually transferring the data.
