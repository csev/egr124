More Significant Features

  • showStatusBar() displays the text in the status bar at the bottom of the browser window

  • goToURL() goes to a specified URL address when the “Go To Lecture” button is pushed

  • The <FORM> element is simple HTML and contains a text area where the user can input from the keyboard and a “pick list” where the user can select the display type from a pull-down list of choices
