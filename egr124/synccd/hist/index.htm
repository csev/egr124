Pre-Cache Slides

This style is the standard style for the Sync-O-Matic 3000 program. It makes good use of the Javascript capabilities of Sync-O-Matic. If browsers fail with this style, then you should try the simpler style.

It is intended that you edit this index.htm file and add your own picture, etc - You may also not want to give all of the above options to the user.

The file inst160.gif in this directory is used to show the instructor picture for audio-only versions of the lecture. You can replace this picture for all lectures using the standard style by replacing \syncomat\style\standard\copy\inst160.htm on the machine that produces these lectures.
