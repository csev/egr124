Notes:

    The word “function” at the top of this slide tells the browser that there is a JavaScript function being defined. The name of the function is “Act()” and the actions it takes are described within the curly braces {}. This function has two variables, identified by the word “var” and named “Text” and “ind”. The “Text” variable stores the string of characters entered by the user. The “ind” variable stores the user’s choice of display type. The function then determines how the user’s text should be presented based on the user’s selection of display types. The user has a choice (from a pull down list that appears on the screen) of displaying their text in a message box, another browser window, or in the status bar at the bottom of the browser’s window. Depending on the user’s choice, the “Act()” function calls another function to display the user’s text.
