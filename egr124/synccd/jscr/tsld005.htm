More Facts

  • JavaScript is NOT Java!

    • Java is a separate application
    • The HTML tag <APPLET> tells the browser where to look for the separate Java file
    • JavaScript is an integrated application
    • JavaScript uses the <SCRIPT> tag in HTML to tell the browser how to interpret the script

Notes:

    Java is a full-fledged programming language that must be compiled (translated) in order to be interpreted by a computer. JavaScript accomplishes many of the same tasks Java is used for, but is not compiled. Instead, it is downloaded by the user’s Internet browser and run on their computer (the client).
