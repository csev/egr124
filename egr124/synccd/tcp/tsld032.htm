The Domain Name System

  • In order to make the use of network addresses more convenient and flexible, each system connected to the Internet also has one or more logical addresses.

  • For example: clunix.msu.edu

  • Unlike IP addresses, the domain address have no routing information - they are organized based on administrative unit

  • There are no limitations on the mapping from domain addresses to IP addresses
