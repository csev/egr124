Definitions

  • URL = text string that specifies a unique document

  • “Home Page” = beginning of a particular Web document or collection of documents

  • Web browser = a client program for navigating the Web, e.g.:

    • Netscape
    • Microsoft Internet Explorer

  • HTML = the language used to compose documents for Web delivery

  • HTTP = the protocol used by Web servers to deliver requested documents to Web browsers
