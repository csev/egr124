Notes:

    This is an example of a screen capture of the program called elm. This program is currently in use in the Pilot project at MSU where 60000+ faculty, staff, and students read E-Mail using UNIX servers.
    Another popular text oriented mail reader is called “pine”.
    While these types of interfaces are not flashy and can be difficult to learn, they can be used from any type of computer including a handheld computer with a modem.
    Often text-based interfaces such as these are very effective for persons with sight limitations.
