Key JavaScript Facts

  • Embedded in HTML

    • JavaScripts are downloaded by the browser as part of the webpage
    • Can also be separate, executable files, but that’s rare

  • Depends on the browser to run the applications (“client-side scripting”)

Notes:

    Client-side scripting means that the JavaScript code is downloaded by a user’s Internet browser and then interpreted by that browser on the user’s computer, which is called the client. This is generally faster than server-side applications, which must be run on the computer where the files themselves originate (called the server). In webpages, the server is the computer where the files for the webpage are stored.
