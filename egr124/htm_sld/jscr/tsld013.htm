<SCRIPT> Tags

  • <SCRIPT> tags

    • These are the HTML identifiers that tell the browser that the page is using a scripting language.
    • The “Language=JavaScript” identifier tells the browser that JavaScript is the specific language being used.

Notes:

    The <SCRIPT> tag is an HTML coding element that tells the user’s Internet browser to interpret the enclosed code as a script. The “LANGUAGE” modifier is used to identify a specific type of script. For example, in a webpage incorporating JavaScript code, the tag <SCRIPT LANGUAGE=“JavaScript”> is used to alert the browser that the HTML document contains JavaScript code.
