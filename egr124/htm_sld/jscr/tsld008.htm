JavaScript and HTML

  • Two ways to incorporate JavaScript

    • Include the script within <HTML> document
    • Have a separate file (.js extension) that is called from your <HTML> document (only available in Netscape 3.0+)

  • JavaScript separated into two parts of the <HTML> document

    • Functions are defined in the <HEAD>
    • Functions are called in the <BODY>

Notes:

    A function is a defined set of actions that the computer takes when the function is called from another part of the program. In JavaScript, a function’s name and the actions it takes are defined in the <HEAD> of the HTML document. These functions can then be called from the <BODY> of the document and the computer will carry out the instructions defined in the <HEAD>.
