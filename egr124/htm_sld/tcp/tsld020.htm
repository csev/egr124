IP Addresses and Network Numbers

  • A range of IP addresses is assumed to be on a single “physical network” from the point of view of the Internet

    • Class A: 1-127.*.*.*
      • 127 networks, 16 Million hosts per network
      • 35.8.2.42 serv1.cl.msu.edu
      • 36.56.0.10 stanford.edu
    • Class B: 128-191.0-255.*.*
      • 16,065 networks, 65,025 hosts per network
      • 141.210.10.11 mars.acs.oakland.edu
    • Class C: 192-223.0-255.0-255.*
      • 2,015,775 networks, 255 hosts per network
      • 198.109.162.6 www.tcimet.net
