PPT Slide

    To find the proper location to use when setting Editor Preferences for your system, use ftp as if you were going to manually upload a file to the web space as follows:
    ftp cps.msu.edu
      Connected to cps.msu.edu.

      220 arctic FTP server (UNIX(r) System V Release 4.0) ready.
      Name (cps.msu.edu:crs): crs
      331 Password required for crs.
      Password:
      230 User crs logged in.
      ftp> cd web (or whatever directory is appropriate)
      250 CWD command successful.
      ftp> pwd
      257 "/user/crs/web" is current directory.
      ftp> bye
      221 Goodbye.
    The setting in preferences would be:
    ftp://cps.msu.edu/user/crs/web/
