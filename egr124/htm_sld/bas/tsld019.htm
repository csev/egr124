Command Mode FTP

  • open pilot.msu.edu

  • cd dir

  • lcd directory

  • put localfile [remotefile]

  • get remotefile [localfile]

  • ascii (use for .html .txt .c files)

  • binary (use for .gif .jpg .doc .zip files)

  • mput *.c

  • pwd

  • bye
