Putting a file on the Web

  • To publish a file, you must get the file into your web directory (typically with FTP)

  • The name of the file becomes the last part of the URL

    /afs/msu/user/c/r/crs/web/crs.gif

    http://www. msu.edu/user/crs/crs.gif
