Notes:

    Telnet connects to a system in a “non-graphical” mode. When using telnet, you communicate with the system through the keyboard and see the results displayed in text on the screen. Often you will connect to a system which is running the UNIX operating system. UNIX is a command-mode operating system with commands such as ls or date. Users familiar with MS-DOS can usually pick up UNIX pretty easily.
    The Pilot system at MSU is run using a number of UNIX systems but you never get to run UNIX commands because you are automatically placed into a menu system specifically designed for Pilot.
