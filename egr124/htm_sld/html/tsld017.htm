Image Attributes

  • HEIGHT=n, WIDTH=n

    • Allows browser to preallocate space for the image
    • Speeds downloading
    • Modem friendly
    • Use with caution

  • ALT=txt

    • Useful for blind users with “talkers”
    • Also useful for slow modem users

  • ALIGN= [TOP, BOTTOM, RIGHT, MIDDLE]
