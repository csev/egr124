HTML Tips (Cont’d)

  • Consider low-bandwidth users

    • Use thumbnail images that link to larger images
    • Give file sizes on links to large documents
    • Minimize number of separate inline images in a document

  • Consider Lynx users:

    • Use the ALT construct:
    • <IMG SRC=“flower.gif” ALT=“[Yellow Rose]”>
    • Offer menus as an alternative to any image maps
