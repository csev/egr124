It is very important that you have properly configured computer equipment available to view the lecture material. We have found that the first few weeks of the course are very bad experiences unless we debug the technology in advance.

This class will not "start at the very beginning". For example this course does not include topics such as:

  • Purchasing a computer
  • Selecting an Internet Service Provider
  • Setting up your internet connection
  • Installing the internet software
It is asumed that you have accomplished this before you start the class. You may need to install and configure the RealPlayer software which allows you to view the lectures.

This is my rough checklist of what is needed to take the class:

  • Internet access is available
    • Dial-up is sufficient
    • America On-Line is sufficient
  • Sufficient Hardware
    • Windows-95 computer with sound capability and 8MB memory (Pentium Preferred)
    • Macintosh capable of playing RealAudio or RealVideo
  • A suitable web browser must be available
    • A Netscape 3.0 or higher
    • Microsoft Explorer or higher
    • AOL 3.0 for Windows-95 (Not just AOL 3.0)
  • The RealPlayer or RealAudio plugin is properly installed
  • E-Mail addresses which the students are reading regularly

There is sufficient material and on-line documentation on the preview web site http://www.vu.msu.edu/preview/egr124/ to completely setup and test the software configuration before enrolling. While setting up your hardware and software, the course staff will respond to E-Mail questions at egr124@vu.msu.edu

It is probably best not to enroll until you have a working system. We have found that the first week of the class is the wrong time to try to debug your hardware and/or software.

If you have any questions, please call me at 517-353-2268 or send email to crs@egr.msu.edu

Dr. Chuck
